How to Use:

    HTML Structure: The HTML provides the basic structure with the game-container for the text, the modal and modal-content for the popup, and includes the necessary CSS and JavaScript files.

    Japanese Text (japaneseText.js): This file should export a string containing the Japanese text for your game.  Example:

// japaneseText.js
export default `
問題用紙を　開いて ください。
どこかへ 行きたい。
`;

Tooltips (wordTooltips.js): This file should export an object where the keys are the words and the values are their tooltips. Example:

// wordTooltips.js
export default {
    "問題用紙を": "Exam Paper",
    "開いて": "Open (te-form)",
    "どこかへ": "Somewhere"
};

Vocabulary (vocabulary.js): This file should export the vocabulary data in the format your openModal function expects. Example:

// vocabulary.js
export const vocabulary = {
    "vocabulary": [
        {
            "title": "問題用紙を",
            "content": {
                "definition": {
                    "meaning": "Exam Paper"
                },
                "example": {
                    "phrase": "問題用紙を もらいました。",
                    "translation": "(I received the exam paper.)"
                }
            }
        },
        // ... more entries
    ]
};

CSS (styles.css):  You'll need to define the CSS styles for the .tooltip, .tooltiptext, .modal, .modal-content, .close-button, etc., to control the appearance of your tooltips and modal.
